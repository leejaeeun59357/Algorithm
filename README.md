## ✏️ 목적

- 코딩테스트를 위한 공부 기록 남기기 Repository

## 📝 Git Flow
1. 이슈 작성
  - Title
    - 코딩테스트 문제 대제목
    - ex) [코딩테스트 12차]
 
2. Assignees
    - 본인 선택

3. Project
    - branch 생성
    - origin branch를 기반으로 아래와 같은 이름으로 생성한다.
        - feature/작업자 #이슈번호
    - 2번에서 생성한 branch를 checkout 후 push한다.

4. Outro
    - PR을 Merge 한 후 Branch를 삭제하고 Issue를 Close 한다.
